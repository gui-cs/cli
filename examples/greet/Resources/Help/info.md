# info

Display application information in a TUI markdown viewer.

[Back to main help](help:help)

## Usage

```
greet info
greet info --cat
```

## Description

The `info` command shows version, features, and usage information
for the greet application. When run without `--cat`, it launches
an interactive TUI viewer. With `--cat`, it renders the content
as ANSI to stdout.

Press **Esc** to quit the viewer.
